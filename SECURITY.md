# Sicherheit

bOtExE Studio läuft **lokal** auf deinem PC. Sensible Daten (z. B. dein TikTok-
Session-Cookie fürs Chat-Senden, OBS-/TTS-Zugangsdaten, API-Keys) werden nur
**lokal** gespeichert und verlassen deinen Rechner nicht — Ausnahme: wenn du
Cloud-TTS-Stimmen nutzt, wird der vorzulesende Text an den jeweiligen Anbieter
geschickt (deine Wahl, mit deinem eigenen Key).

## Eine Sicherheitslücke melden

Bitte melde Sicherheitsprobleme **PRIVAT**, nicht als öffentliches Issue —
besonders alles rund um Login/Session/Keys:

➡️ **[Sicherheitslücke privat melden](https://github.com/dOtExE97/botexe-studio/security/advisories/new)**
(GitHub → Tab „Security" → „Report a vulnerability")

Ich kümmere mich so schnell wie möglich darum. Danke fürs verantwortungsvolle
Melden! 🙏

## Unterstützte Versionen

Es wird jeweils die **neueste Release-Version** unterstützt — halte das
Auto-Update aktiv, damit du Fixes automatisch bekommst.
